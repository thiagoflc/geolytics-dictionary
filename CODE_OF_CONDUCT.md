# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**.

Full text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

Translations (including Brazilian Portuguese):
<https://www.contributor-covenant.org/translations/>

## Enforcement

Reports of unacceptable behavior may be sent to the project maintainer at
**thiagoflc@gmail.com**. All complaints will be reviewed and investigated
promptly and fairly. Maintainers are obligated to respect the privacy and
security of the reporter.

## Scope

This Code of Conduct applies within all project spaces — issues, pull
requests, discussions, commits, and any communication channel associated
with `geobrain` — and also when an individual is officially representing
the project in public spaces.
