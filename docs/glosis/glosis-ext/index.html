Description

Camada de integração que declara owl:imports para módulos GLOSIS (codelists, ISO 28258 backbone, procedures, units), emite os crosswalks SKOS GLOSIS↔CGI e GLOSIS↔INSPIRE Geomorph, e materializa o padrão SOSA Property/Value sobre os campos quantitativos do PVT-Dictionary.
